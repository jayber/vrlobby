Use A-Frame to create your space: A-Frame

Then add the networked aframe component to enable social VR: Networked-Aframe

VRLobby has a Networked-AFrame server which you can use so you don't have to set up or host your own. Add the signalURL parameter to the networked-scene component on your <a-scene> with "https://naf.vrlobby.co" as the value.


I'll be working on adding more support with better social VR functionality. E.G. register and login with VRLobby to always use the same avatar in spaces etc.
